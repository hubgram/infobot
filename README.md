# infobot
The simplest Telegram robot to get information of a message. 

##installation

```sh
git clone https://github.com/i-Naji/infobot.git
cd infobot
sudo pip install -r requirements.txt
# Enter Your Bot Token >> "config.py"
python main.py
```
